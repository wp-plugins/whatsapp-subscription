Simple Mobile number subscription

== Description ==

This plugin will be handy for your users to add their mobile numbers in your site, and you can use this plugin for your whatsapp subscription list. 
This plugin widget will add a form in the sidebar or where ever the shortcode have been added, and users' can add their numbers.

A few notes about the sections above:

*   Shortcode for widget
*   Admin can able to add, edit, delete the mobile number
*   "Tested up to 4.1" is the highest version

== Installation ==
1. Upload `whatsapp-subscription` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
